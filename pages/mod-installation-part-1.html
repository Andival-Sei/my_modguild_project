    SSE Engine Fixes (skse64 plugin)

    Special Intstructions

    Скачать Версия: 6.2 Автор:aers - Nukem - Ryan

    skse64 для исправления различных проблем с движком Skyrim Special Edition.

    Файлы для скачивания
    Main Files(Part 1) SSE Engine Fixes for 1.6.1170 and newerVersion:6.2
    Main Files(Part 2) Engine Fixes - skse64 Preloader and TBB LibVersion:2020.3

    Special Intstructions

    Обратите вниамание, что регистр важен!

    1. После установки откройте и измените следующее: skse\plugins\EngineFixes.toml 
      
	[Patches]
	MaxStdio = 8192
	RegularQuicksaves = true
	[Experimental]
	SaveGameMaxSize = true
    2. Извлеките (Part 2) Engine Fixes - skse64 Preloader and TBB Lib и установите файлы в корневую папку каталога Skyrim Special Edition.
